## 使用方法
---
* 1.下載此專案後，於 Terminal 輸入 https://github.com/whps40333/ALPHA-SHOP.git
* 2.開啟專案資料夾後，於 Terminal 輸入 npm start
* 3.於瀏覽器網址列輸入 http://localhost:3000 後，即可瀏覽專案
* 4.欲停止瀏覽，於 Terminal 輸入 ctrl + c
